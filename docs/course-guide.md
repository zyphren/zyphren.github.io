# 课程指引
这里是所有课程的详细指引，按类别整理好啦～

## 数学类
### 高等数学（上册）
这门课主要学极限、导数、积分，是理工科基础，重点掌握：
- 洛必达法则求极限
- 定积分的几何应用
- 微分方程解法
- 👉 网课推荐：[未央指南](https://weiyangxuexi.github.io/courses/)  # 可加多个链接


### 线性代数
核心内容是矩阵运算和向量空间，考试常考：
1. 矩阵的逆和秩的计算
2. 特征值和特征向量
3. 二次型标准化

## 物理类
### 大学物理（力学）
重点章节：
- 牛顿三大定律（解题核心）
- 功和能的转化
- 刚体的转动

### 电磁学
难点是电场和磁场的综合应用，建议：
- 先吃透高斯定理
- 多做电磁感应的习题

## 通选课
### 中国传统文化
课程内容：
- 儒家思想演变
- 传统节日的由来
- 书法/国画入门

### 计算机通识
适合零基础，主要讲：
- 计算机基本组成
- 简单的Python入门
- 办公软件进阶技巧
