

## 雅思备考时间线规划（建议3个月）

### 第一阶段：基础夯实与框架建立（第1-4周）

**目标**：建立正确的备考认知，掌握各科核心方法论，打牢语法和词汇基础。

| 科目 | 核心任务 | 推荐资源与方法 | 每日/每周建议 |
| :--- | :--- | :--- | :--- |
| **通用** | 建立写作框架 | **Simon Task 1 & Task 2 笔记**（精读） | 第1周完成，理解结构即可 |
| **写作** | 学习段落展开方法 | **Simon视频课程**（B站）：重点看Idea-Explain-Example结构 | 每周2-3小时 |
| **语法** | 查漏补缺，解决长难句问题 | **B站：从句合集** 或 **刘晓燕考研英语语法** | 每天30分钟，持续2周 |
| **词汇** | 积累雅思核心词汇 | **欧路词典“雅思词汇加强版”**，目标掌握60% | 每天50-100词，利用碎片时间 |
| **听力** | 培养语感，熟悉题型 | 听英文歌/看美剧；**真题**Section 1-2练习 | 每天30分钟泛听 + 1套Section练习 |
| **阅读** | 了解题型，练习基础定位 | **真题**限时练习（不严格限时），熟悉顺序出题原则 | 每2-3天1套 |
| **口语** | 了解评分标准，学习答题框架 | **Simon口语视频**，了解Part 1/2/3的答题逻辑 | 每周2-3小时 |

**本阶段结束自检**：
- [ ] 能清晰说出大作文的4段式结构
- [ ] 知道小作文Overview要写什么
- [ ] 语法错误明显减少
- [ ] 听力填空题预判词性有意识

---

### 第二阶段：专项突破与技巧打磨（第5-8周）

**目标**：分科强化，掌握各题型技巧，开始口语题库准备和写作输出练习。

| 科目 | 核心任务 | 具体技巧与方法 | 练习量建议 |
| :--- | :--- | :--- | :--- |
| **听力** | 突破选择题和地图题 | 1. **预判词性**（an后填名词）<br>2. **数字题用纸笔记**<br>3. **选择题注意转折词**（but后往往是答案）<br>4. 针对性强化薄弱题型 | 每天1套真题，精听错题部分 |
| **阅读** | 严格限时，攻克难题型 | 1. **前两篇各16分钟，最后一篇留足时间**<br>2. **利用顺序出题原则定位**<br>3. **T/F/NG“答案分布”技巧**（慎用，作为最后手段）<br>4. **机考用高亮功能** | 每2天1套真题，严格计时 |
| **写作** | 开始动笔，形成个人模板 | 1. 按**Simon结构**写小作文（4段式）<br>2. 按**4段式**写大作文<br>3. **每段用Idea-Explain-Example展开**<br>4. 写完后**对照9分范文修改** | 每周：2篇小作文 + 2篇大作文<br>（建议找人批改或反复自改） |
| **口语** | 过题库，准备素材 | 1. **雅思哥**过Part 1/2/3题目<br>2. **每个Part 2话题准备关键词**，联系个人真实经历<br>3. **录音回听**，发现问题 | 每天30-60分钟，录音练习 |
| **词汇** | 积累写作/口语话题词汇 | 1. **高分替换词视频**（B站），学习分类替换<br>2. 按话题整理（教育、环境、科技等） | 每天30分钟 |

**本阶段重点提醒**：
- 写作的**Task Response（精准回应题目）** 是提分关键，每写完一篇都要检查是否紧扣题目
- 口语不要背诵，要练“有框架地自然表达”
- 机考同学开始熟悉**新东方机考模拟网站**的界面和功能（高亮、复制粘贴）

---

### 第三阶段：全真模拟与冲刺（第9-12周）

**目标**：全科模拟，查漏补缺，保持状态，稳定发挥。

| 科目 | 核心任务 | 冲刺技巧 | 模拟安排 |
| :--- | :--- | :--- | :--- |
| **全科** | 整套模考 | 严格按照考试时间（听力40min + 阅读60min + 写作60min）进行**完整模考** | 每周2-3次完整模考 |
| **听力** | 保持手感，巩固错题 | 回顾之前错题，分析错误类型（拼写、定位、同义替换） | 隔天1套 |
| **阅读** | 稳定速度，减少失误 | 1. 严格按时间分配<br>2. 填空题直接**复制粘贴**<br>3. 不确定的T/F/NG可最后用“分布规律”辅助判断 | 隔天1套 |
| **写作** | 形成稳定模板，控制时间 | 1. **严格按10+5+20+5时间分配**<br>2. Overview和Conclusion快速完成<br>3. 重点检查TR是否跑题 | 每周2套（大小作文一起写） |
| **口语** | 模拟真实考场 | 1. 找语伴或对着镜子**限时模拟**<br>2. **录像回看**，检查肢体语言和流利度<br>3. 复习Part 2素材，确保每个话题都有话可说 | 每天1次完整模拟（P1-3） |
| **错题回顾** | 整理高频错误 | 建立**错题本**，总结自己的易错点和薄弱环节 | 每周整理一次 |

**本阶段关键提醒**：
- **口语避开换题季**：如果考试在1/5/9月，提前确认题库是否稳定
- **机考适应**：确保打字速度，熟悉屏幕阅读和标记功能
- **心态调整**：模考分数波动正常，重点放在“查漏补缺”而非焦虑

## 各科经验速查卡（精华提炼）

### 听力
- **预判词性** → 填空不慌
- **数字用笔记** → 机考防错
- **转折词是信号** → 选择题避坑
- **同义替换是核心** → 多积累

### 阅读
- **前两篇16分钟** → 最后一篇不慌
- **顺序出题定位** → 找不到先跳过
- **答案分布规律** → 仅作最后手段
- **机考高亮+复制粘贴** → 效率翻倍

### 写作
- **结构是骨架**：Simon的4段式（Task1/2通用）
- **Overview是小作文灵魂**：总结2个最显著特征
- **Idea-Explain-Example**：段落展开黄金公式
- **TR是7分门槛**：精准回应题目，不跑题
- **看9分范文**：模仿逻辑，而非7分

### 口语
- **避开换题季**：1/5/9月慎重
- **过题库**：雅思哥过Part 2，准备关键词
- **录音+回听**：自检问题最有效
- **真实经历**：编不如真，有真情实感
